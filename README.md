# Financial Analysis & Valuation — Tata Motors & Reliance Industries

## Overview
A 5-year comparative financial analysis of two NSE-listed companies using ratio analysis, trend analysis, and DCF valuation. The study evaluates profitability, liquidity, and solvency metrics and presents findings through interactive dashboards and financial reports.

## Companies Covered
- **Tata Motors Ltd.** (NSE: TATAMOTORS)
- **Reliance Industries Ltd.** (NSE: RELIANCE)

## Period Covered
FY2019 – FY2024 (5 Fiscal Years)

## Tools & Technologies
- **Microsoft Excel** — DCF model, ratio analysis, trend analysis, dashboards
- **Data Sources** — Annual Reports, BSE/NSE filings, Screener.in

## Methodology
| Technique | Description |
|-----------|-------------|
| Ratio Analysis | Profitability, liquidity, solvency, and market ratios |
| Trend Analysis | 5-year revenue, margin, and earnings trends |
| DCF Valuation | FCFF-based model using WACC and terminal growth rate |
| Dashboard | Visual summary of all findings |

## Key Findings

### Tata Motors
- 5-Year Revenue CAGR: **+11.4%**
- DCF Intrinsic Value: **₹842** vs CMP ₹775 → **Undervalued (8.6% upside)**
- ROE: 18.3% | Net Margin: 4.8% | D/E: 1.92×
- Sharp turnaround from losses in FY20–22 to strong profitability in FY23–24

### Reliance Industries
- 5-Year Revenue CAGR: **+14.2%**
- DCF Intrinsic Value: **₹2,960** vs CMP ₹2,850 → **Fairly Valued**
- ROE: 10.9% | Net Margin: 8.6% | D/E: 0.64×
- Consistent margin expansion driven by Jio, retail, and O2C segments

## DCF Assumptions
| Parameter | Tata Motors | Reliance Industries |
|-----------|-------------|---------------------|
| WACC | 11.5% | 9.8% |
| Terminal Growth Rate | 4.0% | 5.0% |
| Projection Period | 5 Years | 5 Years |
| Intrinsic Value | ₹842 | ₹2,960 |
| Verdict | Undervalued | Fair Value |
